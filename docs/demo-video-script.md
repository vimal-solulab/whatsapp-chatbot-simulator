# Demo Video Script

Use this script for a quick 2-3 minute submission video:

1. Introduce the project and assignment goal.
2. Show the `POST /webhook` endpoint in the code.
3. Start the application.
4. Send `Hi` and show the response `Hello`.
5. Send `Bye` and show the response `Goodbye`.
6. Show the terminal logs for incoming requests.
7. Mention Docker/Render support as the bonus part.
